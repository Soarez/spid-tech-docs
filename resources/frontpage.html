SPiD API Documentation

Learn how to integrate with the SPiD platform. If you're just looking to add Single Sign-On, or sell a suite of products, we've got you covered with guides, articles, working examples and API reference documentation.

Guides

To-the-point technical guides that get you started with the SPiD API.

Getting started

When you have completed this guide, you will have connected your application to the SPiD API and made your first call, verifying that your client ID and secrets are correctly configured.

Implementing Single Sign-On

When you have completed this guide, your users can log into your application via SPiD, and you can access the SPiD API on their behalf.

Using SPiD as a payment provider

When you have completed this guide, your users can purchase products via SPiD, and you can track their purchases in your application to give access to products and pages.
